ใหม่!  เชิญลองใช้ Psiphon 3

Psiphon 3 คืออะไร?

Psiphon 3 คือ เครื่องมือล่าสุดในการเข้าถึงเวปไซต์ที่ถูกสกัดกั้น โปรแกรมดังกล่าวพัฒนาโดย Psiphon Inc. โดยใช้เทคโนโลยี VPN และ HTTP Proxy ซึ่งลูกค้าของ Psiphon 3 จะทราบ access points ใหม่เมื่อเผชิญกับเนื้อหาออนไลน์ที่ถูกเซนเซอร์ เพิ่มโอกาสอีกขั้นในการเข้าถึงเนื้อหาดังกล่าว

ต้องการดาวน์โหลด Psiphon 3?

Click ที่นี่

Psiphon 3 ตอนนี้ใช้ได้กับระบบปฏิบัติการ Windows XP/Vista/7 เท่านั้น โดยรุ่นที่สนับสนุนการทำงานของระบบปฏิบัติการ Mac OS X และ Android จะนำมาเผยแพร่ เร็วๆนี้!

Psiphon 3 อยู่ภายใต้โครงการ open source project คุณสามารถหา source code และข้อมูลของโปรแกรมได้ที่หน้าหลักของโครงการ

Psiphon 3 ใช้งานอย่างไร?

ดาวน์โหลดและรันโปรแกรมการทำงาน เมื่อโปรแกรมเริ่มต้นทำงานแล้ว คุณจะเห็นหน้าต่างแจ้งเตือนความปลอดภัยแจ้งว่า ขณะนี้ คุณกำลังใช้งานผลิตภัณฑ์จาก Psiphon Inc.

หลังจากนั้น Psiphon 3 จะเริ่มต้นทำงานโดยอัตโนมัติ โดยไอคอนของโปรแกรมจะหมุนเพื่อแสดงการเชื่อมต่อ

เมื่อไอคอนสีเขียวปรากฏ แสดงว่าการจราจรทางอินเตอร์เน็ตของคุณกำลังเคลื่อนผ่าน Psiphon 3 เป็นที่เรียบร้อยแล้ว

Psiphon 3 จะหยุดการเชื่อมต่อโดยอัตโนมัติเมื่อปิดโปรแกรม ผู้ใช้สามารถ click ที่ไอคอนของโปรแกรม หากต้องการเชื่อมต่ออีกครั้ง

อย่างไรก็ตาม VPN protocol อาจถูกบล็อคในบางครั้ง โดยหลังจากconnection timeout เป็นเวลา 20 วินาทีแล้ว Psiphon 3 จะทำการค้นหา HTTP Proxy และ SOCKS Proxy ผ่านทาง SSH tunnel โดยอัตโนมัติ ในการทำงานส่วนนี้ การจราจรทางอินเตอร์เน็ตอาจไม่ได้เคลื่อนผ่าน Psiphon 3 ทั้งหมด สำหรับโปรแกรม Psiphon 3 นั้น ตั้งค่า Proxy ให้กับ Windows ซึ่งค่าดังกล่าวเป็นค่าเริ่มต้นของ web browser หลักๆทุกแห่ง
